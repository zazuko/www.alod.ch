# Swiss Archival Institutions alod.ch-site

This provides an [aLOD](http://github.com/zazuko/alod) [alod.ch](http://data.alod.ch). 

## Requirements

You need to have [docker](https://docker.com/) installed.

## Building

    docker build -t sai .
    
## Running

    docker run --rm -p 80:80 sai
